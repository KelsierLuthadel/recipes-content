# Mead Course

## Overview

A foundations course on home meadmaking, the ancient honey-based fermented drink. Equipment, ingredients (honey choice + yeast + nutrient), the basic traditional mead, the fruited (melomel) and spiced (metheglin) variants, and aging-and-bottling. Pairs cleanly with the existing [winemaking](../winemaking/home.md) and [homebrew-beer](../homebrew-beer/home.md) courses.
