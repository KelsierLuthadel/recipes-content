# Spirits

## Overview

An educational overview of how grain alcohol, vodka, whisky, rye and gin are made commercially. The mash-ferment-distil-age process, the difference between pot and column stills, and what defines each category. Reference, not recipe.

For the hands-on, practical counterpart (building a still, running it safely, working recipes for whisky, bourbon, rye, Tennessee whiskey, vodka, grain alcohol, applejack, peach brandy and moonshine) see [American Spirits](../american-spirits/home.md). That course assumes a TTB Distilled Spirits Permit and covers the doing; this tutorial covers the understanding.
